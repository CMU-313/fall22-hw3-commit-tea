# Teedy Applicant Dashboard Information

We build a dashboard page that can be used to view all of the students, their application statuses, and their GPAs. In our model for Teedy as a graduate school application management system, we make each document represent a student. Below, we use the terms document and student interchangeably. Therefore, we added status and GPA as fields for each document. 

When created, the GPA of a document can be specified here. The GPA must fall into one of eight predefined buckets.

When a user clicks on the dashboard page in the navigation bar, they can see a list of all of the documents. The documents are grouped into columns depending on the status of the documents (i.e. all accepted documents would go into a single column). Within each column, each documnent is reprented as a small box with the name and GPA of the student. There are also two buttons, an edit button and a view button.
